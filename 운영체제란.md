# 운영체제란?

- **정의** : 컴퓨터와 하드웨어 바로 위에 설치하여, **사용자 및 다른 소프트웨어와 하드웨어를 연결**하는 소프트웨어 계층
- **목적**

    1) 컴퓨터 시스템을 **편리하게 사용할 수 있는 환경 제공**

    2) 컴퓨터 시스템의 **자원을 효율적으로 관리 → 효율성, 형평성**
